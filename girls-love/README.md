# American Goddessess Story Creator

A small static web app for generating a queer, girls-love inspired sequel concept to Neil Gaiman's *American Gods*.

## About

This project provides a simple interactive page where users can enter character names, setting details, and story tropes. The app then generates a short narrative preview with modern goddess imagery, road trip atmosphere, and tender queer romance.

## Files

- `index.html` — main application markup
- `styles.css` — responsive styling and visual theme
- `app.js` — story-generation logic and UI behavior
- `.gitignore` — ignores common editor and build files

## Usage

1. Open `index.html` in a browser.
2. Enter the character names, setting, and keywords.
3. Select the story tropes you want.
4. Click **Generate Story**.
5. Optionally use **Copy Story** to copy the generated text.

## Deployment

This is a static site and can be hosted on GitHub Pages or any static file server.
